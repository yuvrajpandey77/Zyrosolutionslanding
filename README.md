# Zyro Solutions - AI Receptionist Landing Page

A modern, high-conversion website for Zyro Solutions, an AI automation agency that builds 24/7 AI receptionists for service businesses.

## Design Philosophy

- **Dark, minimal, premium aesthetic** - No neon, no glowing effects
- **Subtle gradients** - Clean, professional color transitions
- **Human-crafted feel** - Premium agency design, not AI-generated templates
- **Strong UX** - High contrast, clean spacing, clear hierarchy

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Modern CSS** - Custom utilities and animations

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

- `/app` - Next.js app directory with pages and layouts
- `/components` - Reusable React components
- `/app/globals.css` - Global styles and Tailwind configuration

## Pages

- `/` - Homepage with all sections
- `/demo` - Demo request form
- `/thank-you` - Confirmation page after form submission

## Features

- Hero section with product mockup
- Services showcase
- Before/After metrics comparison
- Industries served
- Testimonials
- CTA banner
- Full demo request form
- Thank you page
